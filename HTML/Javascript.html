¿Qué es JavaScript?

JavaScript es un lenguaje de programación o de secuencias de comandos que te permite implementar funciones complejas en páginas web, cada vez que una página web hace algo más que sentarse allí y mostrar información estática para que la veas, muestra oportunas actualizaciones de contenido, mapas interactivos, animación de Gráficos 2D/3D, desplazamiento de máquinas reproductoras de vídeo, etc., puedes apostar que probablemente JavaScript está involucrado. Es la tercera capa del pastel de las tecnologías web estándar, dos de las cuales (HTML y CSS) hemos cubierto con mucho más detalle en otras partes del Área de aprendizaje.


¿Cómo utilizar JavaScript con HTML?

Hay dos maneras de añadir JavaScript a HTML: la primera es incrustar el código JS en línea en HTML, mientras que la segunda requiere crear un archivo de JS externo.

1. Introduce código JavaScript en línea en HTML Cuando se trabaja con archivos para Internet, JS necesita ser cargado y ejecutado junto con el marcado de HTML. Puedes hacer esto en línea dentro de un documento HTML.

2. Crea un archivo JavaScript externo Otra forma de cargar JS y ejecutarlo junto con el marcado de HTML es crear un archivo JS externo. El navegador descargará este archivo JS junto al documento HTML cuando un usuario visite la página web.
