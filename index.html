¿Qué es HTML?

HTML son las siglas de HyperText Markup Language (Lenguaje de Marcado de Hipertexto), el lenguaje estándar utilizado para crear la estructura de las páginas web. Se compone de un sistema de etiquetas que organizan el contenido, como textos, imágenes y enlaces, para que los navegadores puedan interpretarlo y mostrarlo de manera organizada y funcional.


Historia HTML

La primera versión de este código apareció en 1991 y fue escrita por Tim Berners-Lee (TBL), siendo poco más que un diseño inicial de 18 elementos, 13 de los cuales aún se conservan. Se lo consideró poco más que una variante del Lenguaje de Marcado General Estándar (SGML) en uso y consistente en un lenguaje de etiquetas, pero ya en 1993 se reconocieron sus virtudes y su potencia en la estandarización de los lenguajes de la Red.


Entonces se inició el trabajo en Html+, una versión más desarrollada, y en 1995 se obtuvo la tercera versión del estándar: HTML 3.0, cuyas sucesivas actualizaciones (3.1 y 3.2) fueron de gran éxito en los primeros navegadores populares de Netscape y Mosaic. En 1997 aparecería el HTML 4.0, ya como recomendación de la W3C, y finalmente en 2006 la versión más reciente, HTML 5.0.


Editores de HTML.

Visual Studio Code: Un editor muy popular que ofrece una gran cantidad de extensiones y soporte para múltiples lenguajes de programación. Su integración con Git y su terminal incorporada lo hacen muy versátil.

Atom: Desarrollado por GitHub, Atom es un editor de texto hackeable que permite personalizar casi todo. Tiene una interfaz amigable y es ideal para proyectos colaborativos.

Sublime Text:Conocido por su velocidad y eficiencia, Sublime Text es un editor ligero que ofrece características avanzadas como la edición múltiple y la navegación rápida entre archivos.

Notepad++:Un clásico entre los editores de texto, Notepad++ es gratuito y ligero. Es ideal para quienes buscan una herramienta simple pero efectiva para editar HTML y otros lenguajes.

Brackets:Este editor está diseñado específicamente para el desarrollo web. Ofrece una vista previa en vivo y es muy útil para trabajar con HTML, CSS y JavaScript.

Adobe Dreamweaver CC: Aunque es un software de pago, Dreamweaver es una opción robusta para el diseño web, ofreciendo herramientas visuales y de código.

Komodo Edit:Un editor gratuito que soporta múltiples lenguajes y ofrece características como la autocompletación de código y la depuración.

VIM:Para los usuarios que prefieren trabajar en la terminal, VIM es un editor de texto altamente configurable que es muy poderoso una vez que te acostumbras a su modo de operación.

¿Que son Etiquetas en HTML?

Las etiquetas HTML son fragmentos de código que se utilizan para estructurar y dar formato al contenido de una página web, indicando al navegador cómo debe mostrar cada elemento. Funcionan como marcadores y generalmente vienen en pares: una etiqueta de apertura y una de cierre, que se escriben entre corchetes angulares (< y >).


¿Que son Atributos en HTML?

Los atributos en HTML son palabras especiales que se colocan en la etiqueta de apertura de un elemento para proporcionar información adicional sobre él y modificar su comportamiento o apariencia. Se componen de un nombre y un valor, y se usan para configurar elementos, ajustar su funcionalidad o definir aspectos como el origen de una imagen (src) o un texto alternativo (alt).


Reglas fundamentales en HTML

Estructura básica:

Todo documento HTML5 debe tener una estructura básica que incluya las siguientes etiquetas: <!DOCTYPE html>: Declara el tipo de documento. <html>: Contiene todo el contenido de la página. <head>: Incluye metadatos, enlaces a estilos y scripts. <body>: Contiene el contenido visible de la página. Etiquetas de apertura y cierre:

Cada elemento HTML debe tener una etiqueta de apertura y una etiqueta de cierre. Por ejemplo: <p>Este es un párrafo.</p> Etiquetas vacías:

Algunas etiquetas no requieren una etiqueta de cierre, como <br/> y <hr />. Estas se consideran etiquetas vacías. Atributos:

Los atributos proporcionan información adicional sobre un elemento. Deben estar entre comillas. Por ejemplo: <img src="imagen.jpg" alt="Descripción de la imagen"> Anidación correcta:

Las etiquetas deben estar correctamente anidadas. Por ejemplo, no debes cerrar una etiqueta antes de cerrar su etiqueta padre: Correcto: <div><p>Texto</p></div> Incorrecto: <div><p>Texto</div></p> Uso de comentarios:

Puedes incluir comentarios en tu código HTML que no serán visibles en el navegador. Se escriben así: <!-- Este es un comentario --> Semántica:

Utiliza etiquetas semánticas adecuadas para mejorar la accesibilidad y el SEO. Por ejemplo, usa <header>, <footer>, <article>, y <section> para estructurar tu contenido. Codificación de caracteres:

Asegúrate de especificar la codificación de caracteres en el <head> para evitar problemas con caracteres especiales: <meta charset="UTF-8">


Etiquetas para hacer un documento HTML.

Para hacer un documento HTML, necesitas las etiquetas básicas: <!DOCTYPE html> para declarar el tipo de documento, <html> para la raíz, <head> para metadatos y <body> para el contenido visible. Dentro de <body>, usa etiquetas como <h1> para títulos, <p> para párrafos, <a> para enlaces e <img> para imágenes para estructurar el contenido.
